# Tools
We use a small set of tools to help us with code quality and maintainability.

## SonarQube
SonarQube doesn't scan the entire repo automatically, yet. The Backend and Frontend are scanned separately and are setup
in SonarQube as separate projects. We also use SonarQube plugins in Intellij to get real-time feedback on code quality.
